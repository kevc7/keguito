# 💌 LoveLetterPage

✨ **LoveLetterPage** es una página estática desarrollada en **JavaScript** con **Angular** 💖. Esta es una carta online de contexto romantico. 

---

## 🚀 Desarrollo y ejecución

### 📡 Servidor de desarrollo
Ejecuta `ng serve` para iniciar un servidor de desarrollo. Luego, ve a `http://localhost:4200/`. La aplicación se recargará automáticamente cuando realices cambios en los archivos fuente.

### 🛠️ Creación de componentes
Usa `ng generate component nombre-del-componente` para generar un nuevo componente. También puedes generar directivas, pipes, servicios, clases, guards, interfaces, enums y módulos con:
```
ng generate directive|pipe|service|class|guard|interface|enum|module
```

### 📦 Construcción del proyecto
Ejecuta `ng build` para construir el proyecto. Los archivos generados estarán en la carpeta `dist/`.

### 🧪 Pruebas unitarias
Ejecuta `ng test` para correr las pruebas unitarias con [Karma](https://karma-runner.github.io).

### 🏁 Pruebas end-to-end
Ejecuta `ng e2e` para realizar pruebas end-to-end con la plataforma que prefieras. Es necesario instalar un paquete que implemente esta funcionalidad.
